Ítems mínimos que deben cumplirse para pasar una issue a **Done**:
- Código/scripts funcionan sin errores.
- Se cumple con los criterios de aceptación.
- Los target de Make correspondientes funcionan.
- Evidencias de seguridad generadas (SBOM/scan/report).
- Documentacion clara y concisa.
- Los flujos de trabajo de CI/CD (GitHub Actions) pasan correctamente.
- Las pruebas de los endpoints (make test) se ejecutan y pasan correctamente.
